# PDF Manager with RAG Assistant

## Overview

**PDF Manager with RAG Assistant** is a personal tool built using Flask, designed for students and professionals who need an efficient way to organize and navigate their notes during critical times like exams. It combines the power of **RAG (Retrieval-Augmented Generation)**, leveraging **Ollama's Llama 3.1**, to answer your queries based on the notes you're currently viewing.

### Key Features
- **Centralized PDF Management**: Keep all your notes in one place.
- **RAG Query Assistant**: Ask questions and get answers directly from your notes.
- **Note Creation and Organization**: Create notes, manage folders, and move notes across folders.
- **Dark Mode**: Perfect for late-night study sessions.

---

## Requirements and Installation 

1. **Ollama**: Install from the [Ollama website](https://ollama.com/).
2. Run the provided `2bat` files and delete them after execution.
